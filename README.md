# msr-challenge
## Installation

```st
Metacello new
	repository: 'github://jesus-acev/msr-challenge:main/src';
	baseline: 'MsrChallenge';
	load
```

## Load Lepiter

After installing with Metacello, you will be able to execute

```
#BaselineOfMsrChallenge asClass loadLepiter
```
